About Me

Work Experience
Undergraduate Research Assistant, University of Waterloo
May - Aug 2017
During my research term, I created a Stata plugin for random forest classification and regression. The backend code is written in Java, which acted as a secondary interface for the Random Forest class methods from the WEKA Project, developed by the University of Waikato. This project was part of Prof. Matthias Schonlau's 5-year NLP research project on semi-automatic analysis of open-ended survey questions.
The plugin is scheduled to be released to all Stata users and researchers by the end of July. Source code for the plugin is available upon request at git.uwaterloo.ca/schonlau/randomforest
